# SkySync - Weather Application

### Developed a fully functional and responsive weather forecasting web application using React. Integrated the OpenWeather API to fetch and display real-time weather data with a clean and modern Material UI interface. The application is deployed on Render, ensuring seamless access and smooth performance across devices.

<h3><img src="./public/weatherAppLogo.png" width="32px"/> <a href="https://skysync-weather.onrender.com">SkySync: https://skysync-weather.onrender.com</a></h3>

### Technologies Used: <br> React.js, JavaScript, Material UI, CSS3, HTML5, OpenWeather API, Render
